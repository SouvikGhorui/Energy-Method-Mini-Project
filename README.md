# Energy-Method-Mini-Project

A mini research project exploring **energy methods** as a tool for proving uniqueness and well-posedness of solutions to Partial Differential Equations (PDEs), with a focus on initial boundary value problems (IBVPs).

This repository collects reference books, presentation slides, research papers, and interactive simulation environments built during the project.

---

## 📁 Repository Structure

```
Energy-Method-Mini-Project/
│
├── PDE Presentation/              # Manim-based animated presentation slides
│
├── damped-wave.html               # Interactive simulation: Damped Wave Equation
├── reaction_diffusion.html        # Interactive simulation: Reaction-Diffusion Model
│
├── Partial Differential Equations An Introduction.pdf   # Reference textbook
├── sharp-hadamard.pdf             # Research paper on Sharp Hadamard well-posedness
```

## 📖 Contents

### 🎞️ PDE Presentation
Animated presentation built with [Manim](https://www.manim.community/), covering:
- Motivation and setup for energy methods
- Derivation of energy identities for IBVPs
- Uniqueness proofs via energy inequalities
- Application to specific PDE models (heat, wave, reaction-diffusion)

### 🌐 Interactive Environments

| File | Description |
|------|-------------|
| `damped-wave.html` | Browser-based interactive simulation for the **damped wave equation**, allowing parameter control and real-time visualization |
| `reaction_diffusion.html` | Interactive environment for exploring **reaction-diffusion dynamics**, with adjustable diffusion and reaction coefficients |

Open these `.html` files directly in any modern browser — no installation required.

### 📚 Reference Material

| File | Description |
|------|-------------|
| `Partial Differential Equations An Introduction.pdf` | Introductory PDE textbook used as primary reference throughout the project |
| `sharp-hadamard.pdf` | Research paper on sharp Hadamard well-posedness, relevant to ill-posed PDE problems and classical well-posedness theory |
